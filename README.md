# Passo a passo back

- Clonar repositório
  >
        git clone {.git-repo}


- Entrar no repositório clonado
  >
        cd {folder}


- Subir os serviços com docker e instalar as dependências
  >
        make install


- Rodas as migrations
  >
        make migration

- Rodas as migrations com seeder
  >
        make migration

- Executar os testes
  >
        make test
